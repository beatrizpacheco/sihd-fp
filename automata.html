Autómata programable

Mediante el autómata programable LOGO! de Siemens hemos programado un sistema de subida y bajada de una persiana de manera manual, mediante interruptores, o de forma automática estableciendo una hora determinada para la subida y otra para la bajada.

El dispositivo se configura mediante un software que viene con el producto y que se instala en un PC.
